# Sales Package

This folder contains the first sales and demo enablement package for `Entruempler Angebotsrechner V1`.

## Recommended usage order

1. `01_demo-content-package.md`
2. `03_customer-one-pager.md`
3. `04_sales-copy-package.md`
4. `05_service-scope-and-pricing.md`
5. `06_sales-offer-structure.md`
6. `02_screenshot-plan.md`
7. `07_packaged-output-guide.md`

## Best use for each file

- `01_demo-content-package.md`
  Demo company profile, realistic example inquiries, and live demo scripts.
- `02_screenshot-plan.md`
  Screenshot order and captions for marketplaces, one-pagers, and portfolio use.
- `03_customer-one-pager.md`
  Ready-to-send one-pager text for local businesses.
- `04_sales-copy-package.md`
  Headlines, subheadlines, pitches, CTA lines, objection handling, and outreach text.
- `05_service-scope-and-pricing.md`
  Clear service scope, boundaries, optional extras, and simple pricing packages.
- `06_sales-offer-structure.md`
  Practical talk tracks for calls, quotes, and customer conversations.
- `07_packaged-output-guide.md`
  Ready combination guide for PDF, internal notes, email, landing page, and proposal use.

## Practical recommendation

For first outreach, use:

1. the short outreach message from `04_sales-copy-package.md`
2. the one-pager from `03_customer-one-pager.md`
3. the short live demo flow from `01_demo-content-package.md`

That combination is the fastest way to move from cold contact to a concrete demo call.
